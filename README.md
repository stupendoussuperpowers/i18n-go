# i18n Puzzles (Go)

Link to puzzles: [https://i18n-puzzles.com/]

To run a specific puzzle: `./mage.sh run <puzzle_number>`

To run a test: `./mage.sh run <puzzle_number> test`

To build: `./mage.sh build`

Solutions in: `pkg/i18n/i18n<puzzle_number>.go`

Contains some horrific date-time code, proceed with caution.
